# Github-Icons

 You can find or add icons in file format (.png/.jpg/.svg)

## Add Icons 
- **Step-1 :** Go to /icons directory
- **Step-2 :** Add your icons in the above file formats
- **Step-3 :** Update the project
  - create your branch
  - add your changes
  - commit with "messages"
  - push your changes
